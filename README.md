# Bewerbungs-Tracker

Ein kleines, persönliches Tool, das ich entwickelt habe, um meine Bewerbungen für Ausbildungsplätze in Deutschland zu verwalten. Das Projekt zeigt, wie ich strukturiert arbeite, und demonstriert grundlegende Frontend-Kenntnisse in React + Tailwind.

## 🧭 Idee
Das Tool erlaubt mir, Bewerbungen hinzuzufügen, den Status zu verfolgen und schnell zu sehen, wo ich nachfassen muss. Es ist bewusst minimal gehalten — genau so, wie ich es Recruitern zeigen möchte.

## 🛠 Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS

## ✨ Features
- Bewerbungen anlegen, bearbeiten, löschen
- Status-Badges (Gesendet, Vorstellungsgespräch, Absage, Antwort erhalten)
- Deutsche UI-Beschriftungen
- Lokale Speicherung (localStorage)

## 📌 Kontext
Ich baue dieses Projekt während meiner Bewerbungsphase für eine Ausbildung als **Fachinformatiker Anwendungsentwicklung**, um Struktur, Sauberkeit und praktisches Denken zu demonstrieren.
